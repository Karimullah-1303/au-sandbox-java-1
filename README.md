# Corporate Sandbox Assessment: SDE-1 (Java/Spring Boot)

## Scenario
You are joining the backend team. The `inventory-service` is failing to start in the CI/CD pipeline. 

## Your Task
1. Run the application.
2. Read the terminal stack trace to identify the application context failure.
3. Fix the dependency injection issue so the application successfully boots on port 8080.
